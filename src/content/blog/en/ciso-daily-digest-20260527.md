## EVERY8D Breach, Lazarus Campaign, Ghost CMS ClickFix, and Widespread Vulnerability Disclosures

- Taiwan's largest OTP messaging platform EVERY8D was hacked, with F-ISAC issuing an orange-level security alert
- North Korean Lazarus Group targeted financial and cryptocurrency institutions using the RemotePE malware
- Ghost CMS SQL injection vulnerability exploited to hack 700+ websites spreading ClickFix attacks; JPCERT issued weekly vulnerability reports covering Drupal, Cisco, Splunk, BIND, Chrome, and more

## 本週活躍威脅

📌 **Microsoft Patches Two Zero-Days: UnDefend and RedSun**

Microsoft fixed two zero-day vulnerabilities — codenamed UnDefend and RedSun — affecting multiple Windows components, with active exploitation reported in the wild.

🔗 **Reference:** [xakep.ru](https://xakep.ru/2026/05/26/undefend-redsun-patch/)

📌 **Laravel Lang Packages Compromised to Distribute Malware**

Compromised Laravel Lang packages on Packagist were found distributing malware to PHP developers, highlighting ongoing supply chain risks in the PHP ecosystem.

🔗 **Reference:** [xakep.ru](https://xakep.ru/2026/05/26/laravel-lang/)

📌 **Megalodon Malware Infects Thousands of GitHub Repos**

The 'Megalodon' malware campaign infected thousands of GitHub repositories in a large-scale software supply chain attack, stealing credentials and injecting malicious code.

🔗 **Reference:** [Dark Reading](https://www.darkreading.com/application-security/megalodon-malware-infects-thousands-github-repos)

📌 **llama.cpp GGUF Parser Critical Integer Overflow Enables Arbitrary Reads in AI Stacks**

Critical integer overflow vulnerabilities were discovered in the llama.cpp GGUF parser, enabling arbitrary memory reads across every local AI deployment using the popular framework.

🔗 **Reference:** [Tech Times](https://news.google.com/rss/articles/)

📌 **AI Chatbot Recommendations Redirect Users to Cryptojacking Malware Sites**

Researchers found that AI chatbot recommendations are being manipulated to redirect users to cryptojacking malware sites, exploiting user trust in AI-generated responses.

🔗 **Reference:** [The Hacker News](https://thehackernews.com/2026/05/ai-chatbot-recommendations-redirect.html)

📌 **Hackers Abuse Google Ads with Claude.ai Share Pages to Distribute macOS Malware**

Threat actors abused Google Ads combined with shared Claude.ai conversation pages to distribute malware targeting macOS users.

🔗 **Reference:** [iThome](https://www.ithome.com.tw/news/)

📌 **Apple Integrates Two Quantum-Safe Algorithms into Core Cryptography Libraries**

Apple added two post-quantum cryptographic algorithms to its OS-level cryptography libraries and introduced formal verification tools.

🔗 **Reference:** [iThome](https://www.ithome.com.tw/news/)

📌 **7-Zip Fixes Critical NTFS Image Bug Allowing Arbitrary Code Execution**

7-Zip released a new version patching a critical vulnerability in malicious NTFS image file handling that could lead to arbitrary code execution.

🔗 **Reference:** [iThome](https://www.ithome.com.tw/news/)

📌 **ExifTool Fixes macOS Command Injection Vulnerability via Malicious Image Metadata**

A command injection vulnerability in ExifTool for macOS allows attackers to execute arbitrary commands through crafted image metadata — update immediately.

🔗 **Reference:** [iThome](https://www.ithome.com.tw/news/)

📌 **Taiwan MODA Tests 4 Chinese Apps: AutoNavi Found More Dangerous Than Google Maps**

Taiwan's Ministry of Digital Affairs tested four Chinese apps for security, finding AutoNavi (Gaode) maps posed greater privacy risks than Google Maps.

🔗 **Reference:** [iThome](https://www.ithome.com.tw/news/)
